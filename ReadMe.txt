【 概要 】
「ExClip」はクリップボードのデータを取り出すためのソフトです。
例えば、「PrintScreen」や「Alt + PrintScreen」でクリップボードにウィンドウなどをキャプチャーしたとき、画像ファイルにするには従来だとペイントソフトを開いて[編集][貼り付け]から[ファイル][名前を付けて保存]などという作業が必要でしたが、「ExClip」では起動するとすぐにダイアログが開きbmp,jpg,png,gifのどれかを選んですぐにファイルに保存できます。
他には、ファイルなどをクリップボードにコピーしているときは、コピーファイルリストを取得・保存でき、テキストがクリップボードにあるときも手軽に保存できます。

【 特長 】
◆クリップボードに貼り付けた画像を手軽にファイル保存できる。起動したらすぐに保存ダイアログが開くので、後は画像フォーマットを選んで保存するだけです。
◆クリップボードに貼り付けたコピーファイルリストを手軽に取得・保存できる。フォルダ内のファイルリストを取得するときにも便利。
◆クリップボードに貼り付けたテキストを手軽にファイル保存できる。
◆シンプル操作。データを取り出したいときに起動するだけ。用件が終わるとアプリも終了します。
◆今のところ、画像データ、コピーファイルリスト、テキストに対応。

【 インストール 】
ダウンロードしたExClip.zipを解凍してください。
※レジストリは使用しません。

【 アンインストール 】
ExClip.zipを解凍したフォルダを削除して下さい。

【 バージョンアップ 】
解凍した中身をインストールしたフォルダ内に上書きしてください。

【 使用方法 】
ExClip.exeを起動するだけです。
クリップボードに画像データがあるときは画像用の動作をし、その他のデータがあるときはその都度ソフトが自動で判断して動作します。
